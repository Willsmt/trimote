# Produto e UI

- Navegação minimalista e contextual: o menu global mostra só o essencial por papel; ações aparecem no contexto onde são usadas (ex.: após concluir atendimento, link direto pro financeiro). Feature nova mantém ações principais a 1 clique SEM inflar a navegação. Simplicidade é a vantagem competitiva.
- Convenção na mecânica, autoral na identidade: seguir padrões estabelecidos de navegação/interação (menu, login, voltar, formulários); investir originalidade no tom das mensagens, estética e decisões de produto — não em mecânica de UI inventada.
- Stack de UI: Tailwind + shadcn/ui. Landing usa CSS Modules isolados e o token de acento `tekhelet-vivo`.
